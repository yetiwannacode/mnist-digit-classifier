# MNIST digit classifier
A PyTorch-based handwritten digit classifier trained on the MNIST dataset.

## Highlights
- Built and trained a neural network for digit classification
- Achieved ~96% accuracy
- Includes preprocessing, model training, evaluation, and visualization

## Tech Stack
Python, PyTorch, Jupyter Notebook, NumPy, Matplotlib

## What I Learned
- Neural network training loop
- Loss function and optimizers
